## Procedure

1. First, make sure Python is installed on your system. If not, download and install it from the official Python website. I actually installed it from the official website, it was much easier and better.

2. Install the required libraries by opening a terminal or command prompt and running:


3. Prepare a folder named **"known_faces"** and add images of the person you want the system to recognize. I didn’t include mine here because I really didn’t want to.

4. If needed, update the folder path in the `main.py` file so the program can access the images correctly.

5. Run the program by executing the `main.py` file using Python.

6. Allow webcam access when prompted so the system can start capturing video. I used my laptop webcam for this purpose.

7. The program will detect faces in real time and compare them with the stored images. It will label them as either **Known** or **Unknown**.

8. To stop the program, press the **Q** key on your keyboard.
